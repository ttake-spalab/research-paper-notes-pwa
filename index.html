fine-grained Personal Access Token を貼り付けて接続します。

fine-grained PAT の作り方:

  1. GitHub → Settings → Developer settings → Fine-grained tokens
  2. 「Generate new token」を押す
  3. Repository access → Only select repositories で research-paper-notes を選択
  4. Permissions → Contents を Read and write に設定
  5. 「Generate token」で発行し、表示された値を上の欄に貼り付け
